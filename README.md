# Chill Guy Bot — Frontend

The marketing site and command reference frontend for **Chill Guy**, a free Discord economy bot with jobs, gambling, pets, stocks, activities, and more.

**Stack:** React 18 · Vite 5 · React Router v7 · Lucide React · Vercel Analytics · Deployed on Vercel

---

## Pages

| Route       | Component            | Description                  |
| ----------- | -------------------- | ---------------------------- |
| `/`         | `App.jsx`            | Marketing landing page       |
| `/commands` | `Commands.jsx`       | Full slash command reference (84 commands across 10 categories) |
| `/privacy`  | `PrivacyPolicy.jsx`  | Privacy policy               |
| `/terms`    | `TermsOfService.jsx` | Terms of service             |

---

## Features Showcased

- **Global Economy** — Wallet, bank, daily/weekly/monthly rewards, peer-to-peer payments, and net worth tracking
- **Gambling** — Blackjack, roulette, slots, crash, mines, horse races, cockfights, scratchcards, and more (14 games total)
- **Activities** — Hunt, fish, mine, chop, and dig with unique loot tables and rarity systems
- **Pets** — Adopt, feed, play, and evolve pets with hunger, happiness, energy, and health stats
- **Stock Market** — 350+ tickers across 7 real-world exchanges; prices update every 30 minutes
- **Jobs & Career** — Apply for jobs, work shifts, get promoted, build streaks, and prestige for permanent pay multipliers
- **Shop & Items** — Buy tools, consumables, and lootboxes; sell gathered resources for rare loot
- **Lottery** — Daily draws with a global prize pool, ticket buying, and a 5% house cut
- **Stats & Rank** — Profile overview, XP/level progression, achievements, streaks, and daily quests
- **Utility** — Ping, cooldown checks, help command, and full profile reset
- **Administrator** — Developer-only commands for balance, items, XP, and cooldown management
